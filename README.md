# 在中国国内玩转Kubernetes

因为网络速度和限制的原因，如果按照Docker/Kubernetes/Helm等工具的官方文档在中国国内使用，你会遇到种种网络有关的错误。其实，国内厂商和学校已经慷慨的进行了制作了很多镜像，来解决国内使用中的问题。这个项目展示了如何完全在中国国内高速顺畅的使用Kubernetes及有关项目。
